Neural Networks (100 Entries)
Entry: Definition of Neural Networks

Description: Computational models inspired by the human brain, designed to recognize patterns.
Key Components: Neurons, layers (input, hidden, output), weights, and biases.
Entry: Architecture of Neural Networks

Description: The arrangement of neurons in layers that determines how data is processed.
Types: Feedforward, Convolutional, Recurrent.
Entry: Activation Functions

Description: Functions that determine the output of a neuron based on its input.
Examples: Sigmoid, ReLU (Rectified Linear Unit), Tanh.
Entry: Training Neural Networks

Description: The process of adjusting weights to minimize the error in predictions.
Techniques: Backpropagation, gradient descent.
Entry: Convolutional Neural Networks (CNN)

Description: Specialized neural networks for processing structured grid data like images.
Applications: Image classification, object detection.
Entry: Recurrent Neural Networks (RNN)

Description: Neural networks designed for sequential data, capable of retaining information from previous inputs.
Applications: Time series forecasting, natural language processing.
Entry: Deep Learning

Description: A subset of machine learning using multi-layered neural networks to model complex patterns.
Advantages: Can learn from large amounts of unstructured data.
Entry: Dropout Technique

Description: A regularization method to prevent overfitting by randomly dropping units during training.
Benefit: Improves model generalization.
Entry: Transfer Learning in Neural Networks

Description: A technique where a pre-trained network is fine-tuned on a new dataset.
Benefits: Reduces training time, improves performance on small datasets.
Entry: Common Neural Network Models

Description: Popular neural network architectures used for various tasks.
Examples: VGGNet, ResNet, Inception.

Entry: Definition of Machine Learning

Description: A subset of AI that allows systems to learn from data and improve over time without being explicitly programmed.
Key Techniques: Supervised learning, unsupervised learning, reinforcement learning.
Entry: Types of Machine Learning

Description: Categories of machine learning methods based on the type of data and the learning process.
Examples:
Supervised Learning: Uses labeled data.
Unsupervised Learning: Uses unlabeled data.
Reinforcement Learning: Learning by trial and error.
Entry: Supervised Learning Algorithms

Description: Algorithms that learn from labeled training data.
Examples: Decision trees, support vector machines, k-nearest neighbors.
Entry: Unsupervised Learning Algorithms

Description: Algorithms that identify patterns in data without labeled responses.
Examples: K-means clustering, hierarchical clustering, principal component analysis (PCA).
Entry: Reinforcement Learning Algorithms

Description: Algorithms that learn optimal actions through feedback from their environment.
Examples: Q-learning, Deep Q-Networks (DQN).
Entry: Feature Selection in Machine Learning

Description: The process of selecting relevant features for model building to improve performance.
Techniques: Recursive feature elimination, LASSO, feature importance.
Entry: Model Evaluation Metrics

Description: Metrics used to assess the performance of machine learning models.
Examples: Accuracy, precision, recall, F1 score, ROC-AUC.
Entry: Cross-Validation

Description: A technique for assessing how the results of a statistical analysis will generalize to an independent dataset.
Types: K-fold cross-validation, stratified cross-validation.
Entry: Overfitting and Underfitting

Description: Overfitting occurs when a model learns noise rather than the underlying pattern; underfitting occurs when a model is too simple to capture the data.
Solutions: Regularization, using more data.
Entry: Popular Machine Learning Libraries
